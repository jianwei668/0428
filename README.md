# 人才项目资料管理系统 - 静态演示版 V1.2 Final

这是一个纯 HTML / CSS / JavaScript 实现的静态演示版，用于展示“人才项目资料管理系统”第一期基础功能。

## 快速开始

直接双击打开：

```text
index.html
```

## 测试账号

```text
管理员：admin / 123456
普通成员：member / 123456
```

## 主要模块

- 成员管理
- 日程安排
- 培训项目
- 书目库 / 听书
- 读书计划

## 技术说明

- 无需安装依赖
- 无需启动服务
- 不依赖后端接口
- 数据存储在浏览器 localStorage
- 适合原型演示、需求确认、前端页面评审
